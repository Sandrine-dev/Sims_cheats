Première Etape

Votre premier étape est d'ouvrir la console, quand vous êtes en jeux. Il vous suffira d'être dans votre partie et d'appuyer sur les touches Shift(ou Maj suivant votre clavier) Ctrl et la lettre C. Une console apparaitra alors en haut à gauche de votre jeux

Deuxième Etape

Avant de renter certain code dans la console il vous faudra les activer, pour ceci il vous suffira de rentrer le code "TestingCheats true" Après avoir entré la commande une ligne apparaitra en dessous Cheats are enable
